Step-By-Step Instructions for the Uploading of Your CCBill Index Page.


1. Open the zip file and extract the contents.
        - You should now have a folder called "909492-0013" with a folder called "ccbill" inside.
2. FTP to your server.
3. Once you are on your server, go to the directory where all of your "Unprotected" content is or will be.
4. Upload the entire "CCBill" folder up to your server.
5. Point your "join" links to the index.htm file in the ccbill Folder.
6. CCBill will finish the installation from this point on.

If you have any questions or need help with this please contact the CCBill tech department at 800.510.2859,
or email us at tech@ccbill.com.
